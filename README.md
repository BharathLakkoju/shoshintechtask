### Getting Started

- First, clone the repository:

  ```bash
  git clone https://github.com/bharathlakkoju/shoshin-task.git
  ```

- Install the dependencies:

  ```bash
  npm install
  # or
  yarn add
  # or
  pnpm install
  # or
  bun install
  ```

- Run the development server:

  ```bash
  npm run dev
  # or
  yarn dev
  # or
  pnpm dev
  # or
  bun dev
  ```
